JSU Tip API core/tooltip.js

JS API for Tooltip (Tip):
  • CORE FEATURES: HTML/Text Floating Tip and Fixed Tip with Optional Close button
  • OPTIONAL FEATURES: Fixed JS Highlighted Tip showing JS code hightlighted with rainbow format
