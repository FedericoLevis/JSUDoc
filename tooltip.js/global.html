TipFixedGoogleAnal(arObjGoogleAnal, event, objOptopt)

Display in a FixedTip a Table with the Link to Google Analytics .
Parameters:
Name Type Attributes Description
arObjGoogleAnal Array Array of Object that identify the Google Analytics. See Exmple Below
event
objOpt Object <optional>
 Option:
  • szTitle{String} default: 'Google Analytics'
  • bShortUrl {Boolean} [true] Show the colum with ShortUrl
  • bLongUrl {Boolean} [true] Show the colum with LongUrl
  • szHeaderTxt {String}: [DEF_TIP_GOOGLE_HEADER] Message to put before the Table of Link to Analytics
  • szFooterTxt {String}: [DEF_TIP_GOOGLE_FOOTER] Message to put after the Table of Link to Analytics
  • bCloseBtn {Boolean} default: true (if true show a Close Button on the Bottom)
  • iWidth {Number}: TipWidth
  • Default = DEF_TIP_WIDTH = 800
  • iMaxHeight {Number}: Max Height (px) of the div that contain the JS (with autoscroll). Default =0 NO SCROLL
  • tipFixedPos: TIP_FIXED_POS.CENTER,... n default=TIP_FIXED_POS.CENTER

TipFixedJS(jsCode, event, objOptopt)

Display a Fixed Tip with JS Code.
  • a) if prettify/prettify-jsu.js is loaded, the code is higlighted
  • b) if prettify/prettify-jsu.js is NOT loaded, the code is displayed as Plain Text in a TextBox with TipFixedTextBox
Parameters:
Name Type Attributes Description
jsCode String jsCode to display, with \n for newline
event
objOpt Object <optional>
 Option:
- szTitle{String} default: 'JS Source Code'
- iWidth {Number}: Width of the Tip - Default: TIP_DEF_WIDTH (800)
- iMaxHeight {Number}: Max Height (px) of the div that contain the JS (with autoscroll). Default =0 NO SCROLL
- bCloseBtn {Boolean} default: true (if true show a Close Button on the Bottom)
- tipFixedPos: TIP_FIXED_POS.CENTER,... n default=TIP_FIXED_POS.CENTER

TipFixedTextBox(szTxt, event, objOptopt)

Display whatever Text (also HTML TAGs) in a TextBox of a FixedTip. For Example this function is used to display mixed JS and HTML code
Parameters:
Name Type Attributes Description
szTxt String szTxt to display, with \n for newline
event
objOpt Object <optional>
 Option:
- szTitle{String} default: 'Text'
- iColNum{Number} default=100 Number of Column for TextBox
- iRowNum{Number} default=20 Number of Rows for TextBox (if more rows are present, scrollbar will be created)
- bCloseBtn {Boolean} default: true (if true show a Close Button on the Bottom)
- iWidth {Number}: TipWidth - Default = DEF_TIP_WIDTH = 800
- iMaxHeight {Number}: Max Height (px) of the div that contain the JS (with autoscroll). Default =0 NO SCROLL
- tipFixedPos: TIP_FIXED_POS.CENTER,... n default=TIP_FIXED_POS.CENTER
