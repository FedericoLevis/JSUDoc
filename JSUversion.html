1) JSU Version Type

You can download 2 different JSU versions:
  • JSU FULL version: All Features are present and JS Source code is available in clear with comments. Also the JS minified files are available: they can be chosen by changing a single line in plugin/jsu.js
  • JSU FREE version: Validate and JQ Popup Features are not available. Tooltip and GoogleAnalytics have Limitations. All JS Code is obfuscated
