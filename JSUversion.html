JSU - JS Utility Plugin - Download Type

Differences between the possible JSU version to download:

OPTION a) JSU FREE Version - JSU DEMO Version

General:
  • SortTable: Sort HTML Table by clicking column header
  • Blocking Popup: HTML Modal Blocking Popup designed specially for IE : based on ShowModalDialog API that is fully supported only by IE. For Browser not supporting ShowModalDialog Popup will simply display an Alert instead of an HTML Popup
  • Tooltip: Many HTML Tooltips options: Floating/Fixed Tips with HTML, GIF, Video, Code Highlight (JS, CSS, Java, C++, …)
    • Code Highlight is not present. However the API can be used, but Code is simply displayed not hightlighted in TextArea
    • A Link JSU Demo Version to this document is displayed in the TipFix Title
    Example of TipFix Limitation in JSU FREE Version:
  • LoadingDiv: Loading Div to be displayed during Long Operation. Many Options provided: Loading Gif, Title, Message, ElapsedSec…
  • Validate: Validate Items with many constraints/options. Show Validate Errors in Red Items or Section, Popup …
  • JSLog: Log from JS Code into an optional Window: log Text, Object, JSON, Time Elapsed …
  • JQ Popup: HTML Modal Popup API working in all Browser. It can be used instead of JSU Blocking Popup

OPTION b) JSU FULL Version

  • ALL Features are available
  • ALL JS Sources are available in clear with comments. JS are provided both with comment and also in the minified version
  • JS API documentation has links to JS line reference
