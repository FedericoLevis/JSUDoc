1) JSU LoadingDiv

JSU API to display a LoadingDiv during Long Operation; many Options provided: Loading Gif, Title, Message, ElapsedSec,...
Very simple API Interface with only 2 JSU API calls required:
(objOpt) Show the LoadingDiv with optional objOpt
Hide the LoadingDiv (at the end of Loading)

We will show following cases:
  • Default LoadingDiv without using the Optional parameter objOpt
  • LoadingDiv Options using the Optional parameter objOpt to change Gif and to add Title, Cancel Button, ElapsedSec,...
  • Advanced LoadingDiv: For advanced LoadingDiv see SAMPLE_3 in LoadingDiv Sample

1.1) Default LoadingDiv

To add to an HTML Item the Floating Tip you have to call the JSU Tip API in the following events:
  • onmouseover="(tipHTML);"
