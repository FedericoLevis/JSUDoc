1) STEPs to display Google Analytics from your HTML page

We will describe the STEPs to add Google Analytics visualization to Following HTML page example: : https://rawgit.com/FedericoLevis/JSU/master/samples/GoogleAnal/SimpleProduct.html
As displayed, it is a very simple page with:
  • A Download Button, that open the Download Page https://rawgit.com/FedericoLevis/JSU/master/samples/JSUFreeDownload.html
  • 4 Links to Product Item pages; for example https://rawgit.com/FedericoLevis/JSUDoc/master/HTML/Tooltip.html for Product Item 1: Tooltip
Our goal is to add the Visualization of the number of Click that are done in the Download Page and in the Single Product Item pages; each counter must be separated per each page.

To easily identify the required modification:
  • We have enphasized the modification with GOOGLE_ANAL Label into the HTML Code of SimpleProduct.html
  • We have saved the HTML with the required modification into a new HTML page SimpleProductWithGA.html
