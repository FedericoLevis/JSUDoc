1) JSU JSLog API

With JSLog API the developer can insert in the JS some statements that can be displayed into the JSLog Window to help during develop/debug/throubleshouting.
  • The default is that JSLog Window is NOT Visible and the JSLog statements have no effects (and they do not add any meaningful delay to JS execution because they correspond only to the time of an IF statement)
  • Each JSLog statement has a specific JSLOG Level (1,2,4,8,16) and only the Level enabled will be logged in the JSLog Window (e.g. Level 3 enables Level 1 and Level 2)
  • JSLog Window is displayed calling jslog_init()
    with jslog_init() you can explicitly enable a JSLog Level or demand it to an optional jslog URL parameter (e.g jslog=3 enables Level 1 and 2)
