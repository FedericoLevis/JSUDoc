1) JSU Tooltip API

Many HTML Tooltips options, supporting whatever HTML Tag (also GIF, Video, ..):
Tooltip APIs & CSS
JSU File JSU API Item Description
jsu/core/tooltip.js Tip() Show a Floating Tip
UnTip() Hide a Floating Tip
TipFix() Show a Fixed Tip
TipFixCode() Show a Fixed Tip with 1 Section for HightLighted Code
TipFixMultiCode() Show a Fixed Tip with N Section for HightLighted Code
jsu/core/jsu.css TipFix Buttons Buttons to show Tooltips, Tooltip Color, Font...
input.jsuInfo Info Buttton to show a Tooltip
We will describe following samples:
  • Floating Tip with Text/HTML tooltip
  • Fixed Tip: HTML Tip in a Window with an optional Close button
  • Fixed Tip with JS Code: JS Code is displayed in a TextBox
