1) JSU Tooltip

Many HTML Tooltips options, all supporting whatever HTML Tag (also GIF, Video, ..):
  • Floating Tip with Text/HTML tooltip
  • Fixed Tip: HTML Tip in a Window with an optional Close button
  • Fixed Tip with JS Code: JS Code is displayed in a TextBox

1.1) Floating Tip

To add to an HTML Item the Floating Tip you have to call the JSU Tip API in the following events:
  • onmouseover="(tipHTML);"
  • onmouseout="(event);"

1.2) Fixed Tip

To implement a FixedTip Tooltip:
  • Set an input with JSU class like
  • In the onclick call the JSU Tip API (tipHTML,event,objOpt)

1.3) Fixed Tip showing JS Code in a TextBox

To show a FixedTip with JS Code in a TextBox:
  • Set an input with JSU class like
  • In the onclick call the JSU Tip API (JScodeMsg,event,objOpt)
