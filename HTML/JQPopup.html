1) JSU JQ Popup

Many HTML Tooltips options, all supporting whatever HTML Tag (also GIF, Video, ..):
  • Floating Tip with Text/HTML tooltip
  • Fixed Tip: HTML Tip in a Window with an optional Close button
  • Fixed Tip with JS Code: JS Code is displayed in a TextBox

JQ Popup - SAMPLE
SAMPLES DESCRIPTION
  • You can Try the various POPUP_TYPE and than Click the Button Show JQ Popup
  • You can change PopupChoice OPTIONS and than Click the Button Show JQ Popup
  • You can change PopupPrompt OPTIONS and than Click the Button Show JQ Popup
POPUP_TYPE:
PopupChoice Options
OPTION Default Value Description
bChoiceMultiSel false true: Multiple Selection - false: Single Selection
iChoiceMultiSize 10
Number of Item Visible (Only for bChoiceMultiSel=true)
PopupPrompt Options
OPTION Default Value Description
szPromptLabel Label in front of the Prompt Item
szPromptType number Prompt Type:number or string
iPromptMin if number: Min Value - if string: Min Length
iPromptMax if number: Max Value - if string: Max Length
szPromptValue Default Prompt Value to set in the Prompt Item
iPromptWidth Width (px) of the Prompt Item
