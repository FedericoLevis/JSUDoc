1) JSU jquery Popup API

Modal Popup designed for whatever Browser . Not Blocking Code with optional Callback Functions
jquery Popup has very simple but powerfull APIs. For example with one single API Popup() You can display different Popup (Information, Warning, Error, Question,...) with an HTML Message containing whatever TAG (supported also GIF, Video, Embedded HTML page,..).
The Popup presentation (Icon, color, font,..) can be easily changed modifying jsu.css that is provide also with an advanced Preview jquery Popup CSS documentation.
jquery Popup APIs & CSS
JSU File JSU API Item Description
jsu/core/JQPopup/Popup.js Popup() Show different Popup (Information, Warning, Error, Question,...) with an HTML Message containing whatever TAG (supported also GIF, Video, Embedded HTML page,..)
PopupCoice() Specific API to Show a Popup with a ListBox or ComboBox to select one/more items

3) Required
