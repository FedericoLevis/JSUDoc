1) JSU SortTable

This JSU API is used to Sort an HTML table by clicking Column Header; you can add this feature to your existing Table without modifying the existing HTML, but simply adding one JSU API:
  • var cSortTblEl = new    Create a cSortTable instance, that will automatically Sort the HTML table
    cSortTable constructor PARAMETERS
    Parameter Type Description Example
    szTblId String DOM id that identifies the HTML Table 'tbl1'
    arSortCol Array Describe each of the column to Sort [
    {col: 'Country'}, // Default type: SORT_TYPE.STRING
    {col: 'Name'}, // Default type: SORT_TYPE.STRING
    {col:'Date', type: SORT_TYPE.DATETIME},
    {col: 'Amount', type: SORT_TYPE.NUMBER}
    ]
    objOpt Object OPTIONAL Option See SortTable API Documentation
    For the details see jsu/core/cSortTable.js

We will describe following cases:
  • Standard SortTable Options: an example with an HTML Table having a common Layout (e.g. 1 Row for Header,..)
  • Particular SortTable Options for particular Table Layout see SAMPLE_2 in SortTable Sample

1.1) Standard SortTable Options

Follow example show how to add JSU Sort Feature to an HTML Table having a standard Layout (e.g. 1 Row for Header,..):

SortTable SAMPLE - Standard Table Layout
SAMPLE DESCRIPTION
SET Table Rows:
of Sample_1:
Table of this sample has
Table Columns:
  • Country: STRING
  • Name: STRING
  • Date: DATETIME with
  • Amount: NUMBER with Default Locale GroupSeparator and DecimalSeparator
Header and Footer:
  • Table Header has 1 rows (That is JSU cSortTable default)
  • Table Footer has class=footer (That is JSU cSortTable default)
