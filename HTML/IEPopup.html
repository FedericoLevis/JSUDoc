1) JSU IE Popup API

JSU IE Popup is a feature designed specifically for following cases:
  • Modal Blocking Popup with Blocking Code
  • ONLY IE Browser because this features uses IE ShowModalDialog() API that is deprecated or removed by other Browsers
IE Popup has very simple but powerfull APIs. For example with one single API Popup() You can display different Popup (Information, Warning, Error, Question,...) with an HTML Message containing whatever TAG (supported also GIF, Video, Embedded HTML page,..).
The Popup presentation (Icon, color, font,..) can be easily changed modifying jsu.css that is provide also with an advanced Preview IE Popup CSS documentation.
IE Popup APIs & CSS
JSU File JSU API Item Description
jsu/core/IEPopup/Popup.js Popup() Show different Popup (Information, Warning, Error, Question,...) with an HTML Message containing whatever TAG (supported also GIF, Video, Embedded HTML page,..)
PopupCoice() Specific API to Show a Popup with a ListBox or ComboBox to select one/more items
jsu/core/jsu.css JSU IE & JQ Popup CSS Popup Icons, Title Color, Background, Font,...
IE Popup should not be used with browser different form IE. However, if it used with different Browser, IE Popup will still work but with limitation: IE Popup will convert the HTML message into text message and it will display it in a simple alert.
You can easily switch between IE Popup and JSU JQ Popup, that supports all the Browser: in fact the 2 features have the same interface and use the same CSS.
