1) JSU IE Popup

Modal Blocking Popup designed only for IE. Blocking Code, based on IE ShowModalDialog()
Very simple but powerful APIs. For example the single API Popup() allow displaying different Popup with whatever HTML tags (and also Gif, Videos,...):

1.1) IE Popup

IE Popup - SAMPLE
SAMPLES DESCRIPTION
  • You can Try the various POPUP_TYPE and than Click the Button Show IE Popup
  • You can change PopupChoice OPTIONS and than Click the Button Show IE Popup
  • You can change PopupPrompt OPTIONS and than Click the Button Show IE Popup
POPUP_TYPE:
PopupChoice Options for Custom Layout
OPTION Default Value Description
bChoiceMultiSel false true: Multiple Selection - false: Single Selection
iChoiceMultiSize 10
Number of Item Visible (Only for bChoiceMultiSel=true)
PopupPrompt Options for Custom Layout
OPTION Default Value Description
szPromptLabel Label in front of the Prompt Item
szPromptType number Prompt Type:number or string
iPromptMin if number: Min Value - if string: Min Length
iPromptMax if number: Max Value - if string: Max Length
szPromptValue Default Prompt Value to set in the Prompt Item
iPromptWidth Width (px) of the Prompt Item
