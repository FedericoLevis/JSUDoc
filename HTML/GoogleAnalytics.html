1) JSU Google Analytics API

The JSU APIs allow displaying Google Analytics related to the Google Shorten URLs from your HTML pages. These Google Analytics pages are available to whatever user.
JSU Google Analytics APIs & CSS for Google Shorten URLs
JSU File JSU API Item Description
jsu/core/googleAnal.js gaShortUrlList() Show a Box with an User Friendly List of Links to Google Analytics Pages
gaShortUrlPage() Show one Google Analytic Page
jsu/core/jsu.css JSU Google Analytics CSS Classes Buttons to call Google Analytics APIS
These Google Analytics have following features:
  • They are automatically generated when the page is loaded using a Google Shorten URLs; no additional code is required.
  • They have the advantage to be visible to all user and JSU APIs provide the facilities to show them
  • They provide only the basic statistic about the number of time the page has been loaded, distributed by nation and by time (last 2 hours, day,week, month, all_time)

JSU provides also a facility to generate all the standard Google Analytics by adding only one JS code line (see 1.3) Generate Standard Google Analytics)

1.3) Enable Standard Google Analytics

The standard Google Analytics have following features:
  • They provide a lot of useful statistics
  • They have the disadvantage to be automatically visible only to the user that has register them (they require to login). However you can use Google Analytics Reporting APIs to create and embed a dashboard on a 3rd party website.
  • They are NOT automatically generated when the page is loaded:
    • You have to retrieve the TRACKING_ID and the JS Code to initialize Analytics.js (see Setup Analytics Tracking
    • Then you have to add these Code lines to your HTML pages.
