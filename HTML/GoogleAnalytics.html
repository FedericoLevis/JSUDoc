1.1) List of Google Analytics

JSU File JSU API Item

jsu/core/jsu.css JSU Google Analytics CSS Classes
SAMPLEs:
List of Google Analitycs
SAMPLES DESCRIPTION

  • Click the Google Analytics Button to show the List of GoogleAnalytics URL in a TipFixed or JQPopup (Available only with FULL JSU)
  • Click Option Button, change Options and try again clicking GoogleAnalytic buttons
TipFix:
 JQPopup (Available Only with FULL JSU):



Steps to implement an UserFriendly Box with a List of Google Analitycs of Google Shorten URL :
  • Get from https://goo.gl/ the Shorten URL associated to the LongURLs of your pages
  • Replace your LongURL with the ShortURL in the Links to your pages
  • Set an input button with the desired JSU Google Analytics CSS Classes
  • In the onclick event call the JSU GoogleAnalytic API gaShortUrlList()
